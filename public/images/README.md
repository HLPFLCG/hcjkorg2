# HCJK Collection - Image Organization

This folder contains all images for the HCJK Photography website, organized by section.

## Folder Structure

### `/portfolio`
- Portfolio showcase images
- Featured photography work
- Gallery items

### `/services`
- Service-specific images
- Wedding photography examples
- Portrait photography examples
- Event photography examples

### `/about`
- About section images
- Team photos
- Behind-the-scenes content
- Brand story visuals

### `/hero`
- Hero section background images
- Homepage banner images
- Featured hero shots

### `/gallery`
- Instagram feed images
- Social media content
- Additional gallery items

### Root `/images`
- `logo.png` - Main HCJK Collection logo

## Image Guidelines

- **Format**: Use WebP for web optimization, with JPG/PNG fallbacks
- **Naming**: Use descriptive, lowercase names with hyphens (e.g., `wedding-ceremony-1.jpg`)
- **Size**: Optimize images before upload (recommended max 2MB per image)
- **Dimensions**: 
  - Hero images: 1920x1080px or larger
  - Portfolio images: 1200x800px
  - Thumbnails: 600x400px

## Adding New Images

1. Place images in the appropriate subfolder
2. Use descriptive filenames
3. Optimize images for web (compress, resize)
4. Update component imports if needed
